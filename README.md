**Sales Performance Dashboard Analysis Using Power BI**

This project is a Sales Performance Dashboard created using Power BI. It helps analyze sales, profit, and quantity data in an easy-to-understand format. The dashboard allows users to filter data by state and view business performance through different charts and graphs.

The dashboard shows sales by customer segment, product category, sub-category, and city. It helps identify which products generate the highest sales and which cities contribute the most revenue. KPI cards are used to display total sales, total profit, and total quantity at a glance.

The main goal of this project is to convert raw sales data into meaningful insights that can help businesses understand their performance and make better decisions. Power BI's interactive features make it easy for users to explore the data and find useful information quickly.

**Key Features:**

**1. State-wise Filtering**: Interactive slicer to analyze data for specific states.

**2. KPI Cards:** Displays Total Quantity, Total Profit, and Total Sales.

**3. Segment Analysis:** Compares sales across Consumer, Corporate, and Home Office segments.

**4. Category Performance:** Shows sales and profit trends for Technology, Furniture, and Office Supplies categories.

**5. Sub-Category Analysis:** Identifies top-performing products such as Phones, Chairs, Machines, and Accessories.

**6. City-wise Sales Distribution:** Visualizes sales contribution from different cities using a donut chart.


**Tools and Technologies**

1. Microsoft Power BI

2. Data Visualization

3. Business Intelligence (BI)

4. Data Analysis

5. Interactive Dashboards

**Insights Generated**

1. Consumer segment contributes the highest sales.

2. Phones and Chairs are the leading product sub-categories.

3. Office Supplies generate the highest profit among categories.

4. City-wise analysis helps identify major revenue-generating locations.

5. Profitability trends can be monitored to support business decisions.

**Project Objective**

The main objective of this project is to transform raw sales data into meaningful business insights using Power BI visualizations, enabling organizations to make data-driven decisions and improve overall sales performance.
